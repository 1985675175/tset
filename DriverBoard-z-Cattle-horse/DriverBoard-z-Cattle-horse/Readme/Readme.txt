 *                           臻合xy-drive板子说明

 * 
 *********************************************************************************************************
 */

1. CPU使用STM32F407IGT6，主频168MHz；
2. AXI SRAM 不存在，使用最后1KBRAM来模拟；
3. C编译器中的预定义宏 （在工程设置中更改）
	USE_HAL_DRIVER        - 增加这个符号表示使用ST公司的HAL库外设驱动
	VECT_TAB_SRAM         - 增加这个符号表示中断向量表定位在CPU内部RAM （针对在CPU内部RAM运行的工程才需要添加，一般都不用）
	STM32F407xx           - 增加这个符号表示使用STM32F407xx系列芯片
	ARM_MATH_CM4		  - 指明CM4内核数学函数
4. 设置堆和栈的大小：
	Stack_Size      EQU     0x00001000
	Heap_Size       EQU     0x00001000
5. 设置rom和ram的地址与大小：
	ROM: 0x8000000  0x100000
	RAM: 0x20000000 0x1FC00
6. 功能描述：
	1) LED3机器运行状态1s闪烁一次,LED1/LED2电机运行状态800ms闪烁一次；
	2) J2端口UART2为信息LOG，打印一些调试信息；
	3) J4端口UART1为RS485通信接口，与上位机交换数据；
	4) SPI2接spiflash数据存储，为后续掉电存储做准备；
	5) 蜂鸣器报警，用于出现故障时触发，可由门锁、液槽、环境温度、总线电压、上位机触发和解除，另外上位机还可禁止相应通道；
	6) 继电器控制加热，与环境温度配合使用，用于恒温控制；
	7) 使用DMA实现两路电机的梯形控制，带原点检测；
	8) 使用ModBus协议实现通信功能；
